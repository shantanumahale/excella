Use the **build-error-resolver** agent to fix build/type errors.

## Instructions

1. Run diagnostics to collect all errors
2. Categorize and prioritize (build-blocking first)
3. Fix each error with minimal changes
4. Verify fix doesn't break other code
5. Iterate until build passes
6. Do NOT refactor, change architecture, or add features
