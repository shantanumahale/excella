description: Create an implementation plan for a feature or refactor
---

Use the **planner** agent to analyze the request and create a detailed, phased implementation plan.

## Instructions

1. Understand what the user wants to build or change
2. Explore the codebase to identify affected files and patterns
3. Create a plan following the format in the planner agent
4. Break into independently deliverable phases
5. Include testing strategy and risk assessment
6. Wait for user confirmation before proceeding to implementation
